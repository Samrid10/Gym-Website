# Gym-Website

Website Link- https://staticfile-eca05.wasmer.app/
A modern, responsive gym and fitness website designed to help fitness centers build a strong online presence. This project includes essential sections such as membership plans, trainer profiles, class schedules, and contact information, providing a complete digital experience for gym visitors.

The website features a clean UI, mobile-friendly design, and smooth navigation to ensure users can easily explore services, learn about trainers, and join the gym. It can be customized for any fitness brand, personal trainer, or health club.

✨ Features

Responsive and mobile-friendly design

Membership plans and pricing section

Trainer profiles and gym facilities showcase

Class schedules and booking section

Contact form and location information

Modern UI for a professional fitness brand

🎯 Purpose

This project demonstrates how a fitness business can use a website to attract members, showcase services, and build a strong online presence.
